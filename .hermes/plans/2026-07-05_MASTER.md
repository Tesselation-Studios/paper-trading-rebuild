## Phase 1: Simulation Engine Core (Hermes)

Files to build in paper-trading-rebuild:

1. `src/llm_engine.py` — OpenRouter API caller
   - `LLMEngine.decide(tick, signal, journal, portfolio, agent_files) → TraderDecision`
   - Reads API key from env, calls OpenRouter, parses JSON response
   - Fallback: HOLD on any error

2. `src/prompt_builder.py` — Assembles full agent prompt
   - `build_prompt(agent_files, tick, signal, journal, portfolio) → str`
   - Agent files: AGENTS.md, SOUL.md, TOOLS.md, MEMORY.md, skill names
   - Journal: last 10 entries
   - Target: under 3000 tokens
   - Skills referenced by name, not full content

3. `src/simulator.py` — Main engine
   - `SweepRunner.run_sweep(trader, variants, params, market_data) → List[SweepResult]`
   - Runs scenarios in parallel via subagents
   - Writes results to DB
   - `SweepRunner.analyze_results() → List[Hypothesis]`
   - Generates new hypotheses from result patterns

4. `src/hypothesis.py` — Pattern analysis
   - `HypothesisGenerator.analyze(results) → List[Hypothesis]`
   - Finds patterns: "Kairos +0.15 when momentum_threshold > 0.55"
   - Generates new prompt variants building on winners
