# Ask Traders Skill

Query all active trader agents for their current status, open positions, and recent decisions.

## Capabilities
- Collect status reports from all trader agents
- Aggregate position data across traders
- Query recent trading decisions and performance
- Report trader health and heartbeat status

## Usage
- Referenced by orchestrator and researcher agents for cross-trader queries
- Skills are loaded from `~/.openclaw/workspace/skills/skill-ask-traders/`
