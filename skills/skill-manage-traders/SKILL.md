# Manage Traders Skill

Orchestrate the lifecycle of virtual trader agents.

## Capabilities
- Create new virtual trader agents with unique configs
- Restart/reload trader agents with updated parameters
- Monitor trader agent health and heartbeat status
- Coordinate with orchestrator for multi-agent operations

## Usage
- Referenced by the orchestrator agent for trader lifecycle management
- Skills are loaded from `~/.openclaw/workspace/skills/skill-manage-traders/`
